# Support

DisplayHub is free and open source under GPL-3.0-or-later. Optional support/funding channels (such as GitHub Sponsors and an optional Microsoft Store purchase) help fund maintenance, bug triage, and ongoing releases.

## User support

- Use **GitHub Discussions** for setup help, usage questions, and general feedback.
- Use **GitHub Issues** for reproducible bugs and actionable feature requests.

## Bug reports

Please include:

- DisplayHub version
- Windows version
- Repro steps
- Expected vs actual behavior
- Logs/screenshots when possible

## Feature requests

Describe:

- The user problem
- Why current behavior is insufficient
- Proposed behavior

## Security and private contact

- Do **not** post security vulnerabilities publicly in Issues or Discussions.
- Report vulnerabilities privately as described in `SECURITY.md`:
  - GitHub: https://github.com/qlintenFX
