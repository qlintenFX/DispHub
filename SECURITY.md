# Security Policy

## Supported versions

Security fixes are prioritized for the latest code on `main`.

## Reporting vulnerabilities

Do **not** report vulnerabilities in public issues.

Please contact the maintainer privately:

- GitHub: https://github.com/qlintenFX

Include:

- Vulnerability description
- Reproduction steps
- Impact assessment
- Suggested mitigation (optional)

We will acknowledge and triage as quickly as possible.
